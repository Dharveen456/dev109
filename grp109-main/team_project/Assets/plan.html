To-Do List App

Project Description

The To-Do List App is designed to help users organize their tasks efficiently. Users will be able to add, edit, delete, and mark tasks as completed. The app will also support local storage to ensure tasks persist even after the page is refreshed.

Purpose

The main purpose of this project is to practice and apply HTML, CSS, and JavaScript skills in a collaborative development setting. This project will enhance our ability to work with local storage, DOM manipulation, and user-friendly interfaces.

Key Features

  • Add, edit, and delete tasks
  • Mark tasks as completed
  • Save tasks in local storage
  • Drag-and-drop sorting
  • Due date reminders
  • Dark mode toggle
